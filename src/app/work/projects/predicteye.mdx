## Overview

Predictye was a product forecasting platform focused on estimating future value for physical goods using historical pricing patterns and predictive modeling.

The work involved building the forecasting flow, shaping the product architecture, and creating a simple interface for users to input product details and receive predictions.

## What I Worked On

- Designed the end-to-end architecture for combining model predictions with a responsive frontend experience.
- Built a stateless REST API capable of serving predictions with p95 latency under 200 ms.
- Implemented caching and data pipeline improvements that reduced computational overhead by 40%.
- Applied feature engineering and model tuning to improve forecast quality, reaching 75% prediction accuracy.

## Technical Focus

- Machine learning and forecasting workflow
- REST API design for prediction requests
- Frontend integration for real-time results
- Performance optimization and caching

## Outcome

The project resulted in a practical forecasting product with a clear technical foundation and measurable performance improvements.
