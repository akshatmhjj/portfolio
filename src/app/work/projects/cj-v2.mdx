## Overview

Code Journey v2 was the production version of the platform. It expanded from a static learning site into a full-stack product with a React and Redux frontend, a Node.js and Express backend, and MongoDB-backed data storage.

The platform was designed to support structured roadmaps, user progress tracking, and company-oriented learning content.

## What I Worked On

- Reworked the architecture from the original static MVP to a scalable React-based application.
- Built and maintained REST APIs for content and user data using Node.js and Express.
- Designed MongoDB data structures for application content and user progress tracking.
- Implemented CI/CD with GitHub Actions on Vercel for deployment and monitoring.
- Improved frontend performance, reducing FCP from 3.2 s to 1.1 s and API latency from 500 ms to 120 ms.

## Technical Stack

- React.js for the frontend interface
- Redux for state management
- Node.js and Express.js for backend services
- MongoDB for data storage
- Tailwind CSS for styling
- GitHub Actions and Vercel for deployment

## Outcome

The project became a more scalable learning platform with better performance, clearer architecture, and stronger foundations for future growth.
