## Overview

Code Journey v1 was the first version of the platform. It was built as a static website using HTML, CSS, and vanilla JavaScript and focused on presenting web development concepts in a structured way for beginners.

The goal was to make core topics easier to follow without relying on a framework or backend.

## What I Built

- Created structured learning sections covering the internet, frontend fundamentals, backend basics, and how full-stack systems connect.
- Built responsive pages with a simple, readable layout and lightweight interactions for navigation and section transitions.
- Organized content so learners could move from concepts to practical understanding without extra complexity.

## Technical Stack

- HTML5 for semantic content structure
- CSS3 for layout and responsive styling
- Vanilla JavaScript for simple UI interactions
- Git and GitHub for version control and deployment

## Outcome

This version served as the initial MVP for the project. It established the content structure and learning approach that later evolved into a larger, production-ready platform.
